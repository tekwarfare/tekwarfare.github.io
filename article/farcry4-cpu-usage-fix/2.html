FarCry 3 CPU Usage fix

If you've experienced stuttering in either FarCry3, Blood Dragon or FarCry 4 then this guide should help to fix that. The crux of the fix involves changing the processors the game's process is allowed to run on. Since I have a 4-core i5 the only affected core is "CPU3", but the logic can be applied to greater core/thread parts.

The problem can be analyzed using an overlay tool like MSI Afterburner and monitoring the usage or utilization of each core. It can be seen that CPU3 is pegged at 100% (more or less) for the duration of the game and is what leads to the stuttering experience.
